# Output Format

Produce the weekly answer using these sections in order:

```markdown
# Weekly Work Summary
## Main Focus Areas
## Recurring Projects And Themes
## Meetings And Decisions
## Concrete Progress And Deliverables
## Open Threads And Risks
## Manager Email Draft
```

Rules:

- Keep the manager draft concise and directly reusable.
- Call out missing weekdays if the source set is incomplete.
- Distinguish explicit evidence from inference.
- Prefer grouped themes over a day-by-day chronology unless the user explicitly asks for a timeline.
- If the weekly markdown already exists, prefer editing that file instead of drafting a second competing version.
